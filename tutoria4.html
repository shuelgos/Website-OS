Configurar Sistemas Operativos y Gestionar Recursos Tecnológicos

Configurar correctamente un sistema operativo y administrar bien los recursos tecnológicos (hardware, software, redes y dispositivos) es clave para que cualquier entorno informático funcione de forma estable, segura y eficiente. En esta actividad se pone en práctica ese conocimiento mediante un ejercicio concreto: la instalación de un sistema operativo dentro de un contenedor usando Docker.

Sobre la actividad

Qué se busca

  • Demostrar el proceso de configuración de un sistema operativo paso a paso.
  • Mostrar cómo se gestionan los recursos asignados a ese sistema.
  • Documentar el proceso en un video explicativo.

Enfoque elegido

  • Uso de Docker para crear un contenedor.
  • Instalación del sistema operativo dentro de ese contenedor.
  • Configuración y gestión de los recursos asignados al contenedor.
